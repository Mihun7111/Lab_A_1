# 台灣縣市景點地圖 (Taiwan Tourist Map)

這是一個互動式的台灣地圖網頁應用程式，旨在介紹台灣各縣市的特色景點。使用者可以透過點擊地圖或清單選擇，快速瀏覽各縣市的介紹、知名景點及其相關圖片。

## 🌟 專案目的

本專案的主要目的是提供一個直觀且美觀的介面，讓使用者能夠探索台灣的地理分佈與觀光資源。無論是教學用途還是旅遊參考，都能透過此網頁獲得豐富的資訊。

## ✨ 主要功能

### 1. 互動式地圖 (Interactive Map)

- **縣市點選**：在地圖上直接點擊任意縣市區塊，該區塊會變色高亮顯示。
- **資訊卡片**：點選後，右側的資訊面板會即時顯示該縣市的名稱、特色介紹，以及三個推薦景點的按鈕。
- **自動捲動**：點擊縣市後，網頁會自動平滑捲動至下方的圖片展示區。

### 2. 縣市下拉選單 (Dropdown Selection)

- 提供一個下拉式選單，包含台灣所有縣市。
- 選擇縣市後，地圖上對應的區域會同步高亮，並觸發與點擊地圖相同的資訊更新與捲動效果。

### 3. 景點詳細介紹 (Spot Details)

- **景點按鈕**：在資訊面板中，每個縣市提供三個具代表性的景點按鈕。
- **圖片切換**：點擊不同的景點按鈕，下方的「圖片區域」會切換顯示該景點的高畫質照片及說明文字。
- **視覺回饋**：當前選中的景點按鈕會以亮色顯示，讓使用者清楚目前的瀏覽狀態。

### 4. 圖片輪播 (Image Slideshow)

- 頁面底部包含一個精選圖片輪播區。
- 支援**自動播放**功能。
- 提供**左右箭頭**讓使用者手動切換圖片。
- 點擊輪播圖片可連結至相關的旅遊或活動網頁。

### 5. 現代化響應式設計 (Modern & Responsive Design)

- **卡片式風格**：採用柔和的灰白背景搭配圓角卡片設計，視覺體驗舒適且現代。
- **RWD 響應式**：版面會根據螢幕寬度自動調整，在桌機與行動裝置上皆能良好顯示。

## 🚀 如何操作

1.  **開啟網頁**：使用瀏覽器打開 `tw.html` 檔案。
2.  **探索縣市**：
    - 滑鼠移動到地圖上的縣市，會看到黃色的游標懸停效果。
    - 點擊你想查看的縣市。
3.  **瀏覽資訊**：
    - 查看右側面板的說明。
    - 點擊面板中的「景點按鈕」（如：阿里山日出、高美濕地等）。
4.  **觀看圖片**：網頁下方會顯示對應的大圖，再次點擊不同景點可切換圖片。
5.  **重置地圖**：點擊地圖以外的空白區域，可以清除目前的選取狀態。

## 🛠️ 技術架構

- **HTML5 & SVG**：建構地圖向量圖形與網頁結構。
- **CSS3**：使用 Flexbox 佈局、CSS Animation 及現代化的陰影與配色設計。
- **JavaScript (Vanilla)**：
  - DOM 操作與事件監聽 (Event Listeners)。
  - `scrollIntoView` 實現平滑捲動效果。
  - 自動輪播邏輯 (Slideshow logic)。

## 📂 檔案結構

- `tw.html`: 網頁主程式與 SVG 地圖資料。
- `style.css`: 包含所有的樣式設計與 RWD 設定。
- `script.js`: 控制互動邏輯、資料綁定與輪播功能。
- `README.md`: 本說明文件。
